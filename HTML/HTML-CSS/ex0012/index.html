Colocando audio no site

Primeira forma:

Para usar o comando 'audio' é necessario ele conter o parametro 'controls autoplay' para que funcione.
Aceita apenas formatos MP3, WAV, OGG.

Segunda forma:

Para essa segunda forma o audio tem que ter o parametro 'src' e não 'srcset'.
Em 'type' colocamos 'audio/formato do audio'.

use o parametro 'loop' dentro do comando 'audio' para a musica ´da um loop assim que terminar de toca.
